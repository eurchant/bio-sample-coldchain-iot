- 登录页明确提示“演示身份，非后端认证”；管理员、发出方、承运方、接收方均不需要密码或 Token。承运方实际验证只显示任务、监控和路线菜单，追溯与大屏入口被隐藏。

- 管理员实际验证可进入单任务实时大屏，显示 TASK-001 的状态、温湿度、ECharts 历史趋势、箱体/运动状态、告警和后端交接节点；历史仅有一条记录时会显示真实单点，不补造趋势数据；全屏按钮可在进入/退出状态间切换。
- 单任务总览、详情、地图、报告和大屏已统一接入加载、空态、断网/接口失败与重试反馈；网络恢复后仍通过既有轮询和接口刷新数据。

## 联调配置

真实联调仅需创建不提交的 .env.local：

    VITE_DATA_SOURCE=api
    VITE_API_BASE_URL=https://组长提供的地址

页面不会在组件中写死后端地址。Mock 模式只提供查询数据，不模拟任务状态机。
